# Vrinda Store Annual Sales Dashboard

## About the Project

This project is an annual sales analysis dashboard created for Vrinda Store, an online retail business.
The goal of this project is to analyze sales data and present it in a visual and easy-to-understand way using charts and filters.

---

## Project Goals

* Understand monthly sales trends
* Analyze order performance
* Study customer behavior by gender and age group
* Identify top-performing states
* Compare sales channels such as Amazon, Flipkart, and Myntra
* Build an interactive dashboard

---

## Dashboard Overview

The dashboard includes:

* Order vs Sales trend chart showing monthly performance
* Order status distribution highlighting delivery success
* Gender-wise and age-wise customer analysis
* Top five states by sales
* Sales contribution by channel
* Interactive filters for month, category, and channel

---

## Tools Used

* Microsoft Excel 
* Data cleaning and formatting
* Pivot tables, charts, and slicers
* Basic data analysis techniques

---

## Key Insights

* Sales peak during the first quarter, especially in March
* Approximately 92 percent of orders are successfully delivered
* Women contribute the majority of sales compared to men
* Amazon is the highest-performing sales channel
* Maharashtra is the top revenue-generating state
